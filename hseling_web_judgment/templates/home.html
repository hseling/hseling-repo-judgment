Парсер судебных решений

автоматическое извлечение данных из уголовных приговоров


Наши алгоритмы анализируют тексты уголовных приговоров и представляют их в структурированном виде.

А именно:

  • извлекают метаданные:
    • суд и регион
    • дату решения
    • номер дела
    • имя судьи
    • имя обвиняемого
    • статьи, по которым проходит данное дело
  • делят на смысловые части:
    • фабула
    • показания свидетелей
    • описание доказательств
    • размышления судьи

Наш поисковый движок позволяет производить поиск по базе по заданным фильтрам:

  • по региону
  • по году
  • по статье уголовного кодекса

Также вы cможете скачать обработанные документы (или один документ) в формате excel. Чтобы обработать свой документ в формате html, воспользуйтесь кодом и инструкцией по ссылке.

База составлена из документов, представленных на сайте Судебные и нормативные акты РФ, всего на нашем сайте 21,5 тыс. дел за 2018-2019 гг. Проект разработан студентами магистратуры ВШЭ “Компьютерная лингвистика” в 2019-2020 гг. Исходный код доступен по ссылке. Подробнее об алгоритме можно прочитать в статье. Исследование было представлено на постерной сессии конференции AINL 2019.
